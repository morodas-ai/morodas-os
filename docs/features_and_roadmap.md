# NoimosAI級Webアプリ 機能一覧と開発ロードマップ

COO補佐（学徒出陣）が提案し、オジキ（k k）が採用した機能一覧です。
**「停滞検知」**と**「収益トラッキング」**を最重要視し、自律的にオジキの目標（1撃100万円、法人化）をサポートします。

## 優先度：最高（🔴 High）
**オジキの「停滞」を防ぎ、確実に「収益」を可視化・最大化する機能群**

| No. | 機能名 | 概要 | 目的 |
| :--- | :--- | :--- | :--- |
| **1** | **⏱️ 停滞検知アラート** | タスクが一定期間（例: 2日）動いていない場合、自動でアラートを発報。「止まってますよ、どうしますか？」と問いかけ、再始動を促す。 | **オジキの最大の弱点「停滞」をシステムでカバーする。** |
| **2** | **💰 収益化トラッキング** | Note、コンサル、構築代行など、分散する副業収益を一元管理し、月次グラフ等で可視化する。 | **目標（1撃100万円）と現状のギャップを常に意識させる。** |
| **3** | **☀️ 毎朝の「今日やるべき」** | 毎朝、その日の優先タスク（緊急度・重要度に基づく）上位3つと推定時間を提示する。 | **「何からやろう？」という迷い（認知負荷）をゼロにする。** |

## 優先度：高（🟡 Medium）
**業務効率を上げ、機会損失を防ぐ機能群**

| No. | 機能名 | 概要 | 目的 |
| :--- | :--- | :--- | :--- |
| **4** | **🎯 営業トークスクリプト自動改善** | 過去の商談・トーク内容と成約率を分析し、勝ちパターンを抽出してスクリプトを自動アップデートする。 | **年1億セールスの知見を資産化し、属人性を排除する。** |
| **5** | **📊 クライアントパイプライン (CRM)** | リード、商談中、受注、入金のステータスを可視化。「あと何件で目標達成か」を表示する。 | **案件管理の抜け漏れを防ぎ、売上予測を立てる。** |
| **6** | **🚨 競合アラート** | 競合（例: ユニコ、AIインフルエンサー）の新着コンテンツを監視し、対抗策や差別化コンテンツを即座に提案する。 | **市場でのプレゼンス維持と、後追いにならないための対策。** |
| **7** | **📅 法人化カウントダウン** | 4月の法人設立に向けたタスク（定款、登記、口座開設など）の進捗と残日数を表示する。 | **期限がある重要イベントの遅延を防ぐ。** |

## 優先度：中〜長期（🟢 Low/Future）
**組織化・自律化を加速させる機能群**

| No. | 機能名 | 概要 | 目的 |
| :--- | :--- | :--- | :--- |
| **8** | **🧠 思考の癖トレース** | オジキの判断ログ（いつ、なぜ、どう決めたか）を蓄積・分析し、AIが「オジキならこう判断する」と先回りできるようにする。 | **「思考の同期」を実現し、AIの承認精度を上げる。** |
| **9** | **📋 承認待ちキュー** | 複数エージェントからの成果物（記事案、ツイート案など）を優先度順に並べ、ワンクリックで承認/修正指示ができるUI。 | **レビュー負荷を下げ、ボトルネック化を防ぐ。** |
| **10** | **🔄 エージェント間の引き継ぎ** | 例：SEOエージェントが見つけたキーワードを、自動でSNSエージェントに共有しツイート案を作るなど、横の連携を自動化する。 | **サイロ化を防ぎ、情報の相乗効果を生む。** |
| **11** | **🎓 スキルマップ** | オジキの脳内スキルとAIの習得スキルを可視化。AIにまだ足りない知識（ギャップ）を特定する。 | **COOとしてのAIの教育進捗チャック。** |
| **12** | **🛡️ 事業継続ドキュメント (BCP)** | オジキが稼働不能になった際に、サーバー維持、顧客連絡などをAIや第三者が代行するための手順書・キー情報を管理。 | **バス係数1（オジキのみ）のリスクヘッジ。** |
